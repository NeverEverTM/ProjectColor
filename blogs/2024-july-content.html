     It was fairly easier to get it done after organizing through the mess that is the codebase.

Aphid Festival Resort 1.3 Update ¶

    It's been a long workload for the new update, and far more work than I thought,
    however that does not mean it has been the only thing being worked on.
    
    Some small QoL improvements like a change in controls, adding in some key and button shortcuts,
    fixing issues that went unnoticed like light affecting UI (this is why you do proper testing).
    
    

    With each new update, I would like to introduce a few trinkets, recipes, items and structures so content-wise
    the game has a least a lil something every month rather than like what happened with this one,
    starting with this hopefully coming next month one, some initial furniture such as the one you
    see already in-game should be available to mess around with.

    I talked about how a screenshot function was added last month, along with the free camera mode,
    that should allow a better check in things within the resort.
